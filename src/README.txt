#-- SYSTEM REQUIREMENTS --#
-- Mac OS X 10.5+ (Python Pre-Installed)
-- Linux Kernel 2.6+ (Python Pre-Installed)
-- Windows XP, 2003, Vista, 2008, 7
   - Python 2.5+
   - wxWidgets 2.8.11, http://www.wxwidgets.org/
-- wxPython 2.8.11, http://www.wxpython.org/

#-- INSTALLATION INSTRUCTIONS --#
-- Windows
	Install Python, wxWidgets, and wxPython.

-- Mac OS X and Linux
	Install wxPython

-- All Operating Systems
Extract the contents of the package anywhere.  Open a terminal and move to the location where you extracted the contents.  Test each portion of the program with the following commands:

	Command Line Program:
	python wbCLI.py

	Graphical User Interface Program
	python wbGUI.py
